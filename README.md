# pcinfo

Tools to check you memory usage

## Installation

To install this package use command

```bash
npm install -g @catnuxer/pcinfo
```

## User guide

To diplay the list of help command line

```bash
npx pcinfo help
```

To check the process memory usage

```bash
npx pcinfo memUsage
```
